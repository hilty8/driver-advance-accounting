# 画面/機能概要

## O: 運営

- 初期画面: テナント数の推移、監査ログ、延滞管理（要検討）
- テナント管理: テナント別支払金額
- 監査ログ一覧: 50 件ごとのページネーション
- 設定画面: 項目は要検討
- システムログ一覧: エラー通達とステータス付与

## C: 事業者

- 全体画面: 前払い総額（年次）、日報登録（CSV）、承認履歴、回収状況、通知欄
- 利用明細画面: 要検討
- 承認管理: 直近 50 件、承認/非承認（理由必須）
- 設定: 支払日設定（1/5/10/15/20/25/月末、土日祝は前営業日に前倒し）、給与超過許容、上限率
- 通知一覧: 既読処理、古い通知の削除は要検討
  - 仕様は `docs/spec/notifications.md` に集約
- 日報取り込み: CSV 推奨、Web 入力可、編集/削除は理由必須
- ドライバー管理: 登録/編集/論理削除、復活可

## D: ドライバー

- 入金予定の表示（給与/前借り/貸付）
- 直近 5 件の前借りステータス一覧
- 前借り申請
- 通知一覧（古い通知の削除は要検討）
  - 仕様は `docs/spec/notifications.md` に集約

## C/D 共通の設定内容

- C の設定: 支払日設定（1/5/10/15/20/25/月末、土日祝は前営業日に前倒し）、給与超過許容、前借り上限率
- D の設定: 振込先口座情報（任意）
