# ユーザーストーリー（大まかなフロー）

- 導入/初期設定: C が Stripe 連携でアカウント作成
- ドライバー登録: C 登録 → D 招待 → アカウント作成
- 稼働データ取り込み: CSV インポート（採用予定）
- 前借り申請: D 申請 → C 承認/非承認
- 前借り承認/振込: 承認 → 振込 → 振込済み更新
- 回収額調整: C が次回回収額を編集可能
- 給与確定: 締め日で確定し、O に通知
- 月次請求: Stripe Invoice API による請求
- SLA エスカレーション: 30 分前/期限到達/3 時間超過
- 補償金額の貸付: 事故時のマイナス処理
- ID/PW 再発行: メール経由で再通知
